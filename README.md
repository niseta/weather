Para instalar las dependencias del cliente ejecutar,en la raiz del proyecto, el comando "npm client-install" y para las del servidor, "npm server-install".

Para correr el cliente y el servidor ejecutar "npm run dev" en la raiz del proyecto.

Para correr pruebas del servidor ejecutar "npm test". Las pruebas utilizan las bibliotecas Mocha, Supertest y Should.js.
